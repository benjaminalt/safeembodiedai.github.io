# Safe Physical AI Workshop

Website for the 2-Day Workshop on Safe Physical AI at IJCAI/ECAI 2026 in Bremen, Germany.

## Development

This site is built with Jekyll and can be hosted on GitHub Pages.

### Local Development

1. Install dependencies:
```bash
bundle install
```

2. Run the Jekyll server:
```bash
bundle exec jekyll serve
```

3. Visit `http://localhost:4000` in your browser.

### Deployment

The site is automatically deployed to GitHub Pages when pushed to the main/master branch.

## Structure

- `_config.yml` - Site configuration
- `index.html` - Main page content
- `_layouts/default.html` - Page layout template
- `assets/css/style.css` - Styling

## Updating Content

### Adding Speakers

Edit the `#speakers` section in `index.html` to add invited speaker information.

### Adding Schedule

Edit the `#schedule` section in `index.html` to add the workshop schedule.

### Updating Organizers

Edit the `#organizers` section in `index.html` to modify organizer information.
